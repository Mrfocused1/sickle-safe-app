PLACEHOLDER ASSETS

The following files are needed but can be temporary placeholders:

1. icon.png (1024x1024) - App icon
2. adaptive-icon.png (1024x1024) - Android adaptive icon
3. splash.png (1284x2778) - Splash screen
4. favicon.png (48x48) - Web favicon

FOR NOW: Expo will use default placeholder assets if these are missing.
You can add custom assets later.

To generate icons easily:
- Use https://www.appicon.co/
- Or design in Figma/Canva and export
